🟩 Wordle Clone – Угадай слово на 7 языках
Популярная игра Wordle в терминале!
Угадай загаданное слово из 5 букв за 6 попыток.
Получай цветные подсказки: 🟩 зелёный – правильная буква на своём месте, 🟨 жёлтый – буква есть, но не там, ⬜ серый – буквы нет.
Реализована на 7 языках программирования с единым интерфейсом, словарём на русском и английском, статистикой и продвинутыми функциями.

🚀 Возможности
Случайное слово – выбирается из встроенного словаря (100+ слов) или из внешнего файла.

Цветная обратная связь – каждый символ окрашивается по правилам Wordle.

Статистика – количество сыгранных игр, процент побед, текущая серия, максимальная серия.

Подсказка – возможность открыть первую букву (снимает одно очко).

Выбор языка – словарь на русском и английском (переключается опцией).

Сохранение статистики – данные хранятся в файле ~/.wordle_stats.json.

Интерактивный ввод – проверка на допустимые символы и длину слова.

Завершение игры – после победы или поражения показывается загаданное слово.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [язык] [опции]
Аргументы
ru – русский словарь (по умолчанию).

en – английский словарь.

-h, --help – справка.

-s, --stats – показать статистику и выйти.

-r, --reset – сбросить статистику.

Примеры
bash
# Игра на русском
python wordle.py ru

# Игра на английском
python wordle.py en

# Показать статистику
python wordle.py --stats

# Сбросить статистику
python wordle.py --reset
Управление
Введите слово из 5 букв и нажмите Enter.

После каждой попытки вы увидите цветные подсказки.

Нажмите q в любой момент для выхода.

🧠 Логика игры
Генерация слова – случайное слово из словаря (5 букв).

Ввод игрока – проверка длины (5 символов) и допустимых букв (только кириллица или латиница).

Сравнение – для каждой буквы определяется:

🟩 – буква есть на той же позиции.

🟨 – буква есть, но позиция не совпадает.

⬜ – буквы нет.

Подсчёт попыток – после каждой попытки количество оставшихся уменьшается.

Окончание – при угадывании слова или исчерпании попыток игра заканчивается, показывается правильное слово.

Статистика – обновляется после каждой игры.

✨ Дополнительные фичи
Подсказка – команда ? открывает первую букву (только один раз за игру).

Два словаря – русский и английский, легко расширяются.

Статистика – сохраняется в JSON-файле в домашней папке.

Цветной вывод – использует ANSI-коды, совместим с большинством терминалов.
